tmux

cheat sheet
prefix = Ctrl+B
Sessions
Ctrl+B dDetach from session
Ctrl+B $Rename current session
Ctrl+B sList / switch sessions
Ctrl+B ( )Previous / next session
Session CLI
tmuxNew unnamed session
tmux new -s mainNew named session
tmux lsList sessions
tmux aAttach to last session
tmux a -t mainAttach to named session
tmux kill-session -t mainKill named session
Windows (tabs)
Ctrl+B cNew window
Ctrl+B ,Rename window
Ctrl+B &Close window
Ctrl+B 1-9Jump to window by number
Shift+← Shift+→ no prefixPrevious / next window
Panes (splits)
Ctrl+B |Split vertical
Ctrl+B -Split horizontal
Ctrl+B xClose pane
Ctrl+B zZoom pane (toggle fullscreen)
Ctrl+B SpaceCycle pane layouts
Ctrl+B { }Swap pane left / right
Navigation
Alt+←↑↓→ no prefixMove between panes
Ctrl+B qShow pane numbers (then press number)
Ctrl+B ;Jump to last active pane
Ctrl+B oCycle through panes
Scroll & Copy
Ctrl+B EnterEnter copy mode
↑↓ or PgUpScroll (while in copy mode)
vBegin selection (vi mode)
yCopy selection & exit
Ctrl+B ]Paste buffer
q or EscExit copy mode
Misc
Ctrl+B rReload config
Ctrl+B ?Show all keybindings
Ctrl+B :tmux command prompt
Ctrl+B tShow clock
Ctrl+B ~Show messages log
no prefix = works without Ctrl+B (custom binding)
Ctrl+B = press prefix first, then the key
